# Shamshir Trading Engine

Algorithmic trading engine with prop-firm-oriented risk management. .NET 10, C# 13.

## Quick Start

```powershell
# Build
dotnet build

# Run tests (69 unit + 3 integration + 2 simulation = 74 total)
dotnet test

# Run backtest (reads CSV from tests/data/)
dotnet run --project src/TradingEngine.Host

# View results in browser
dotnet run --project src/TradingEngine.Web
# → http://localhost:5000
```

## Generate Test Data

Test CSV data files are generated by running the simulation tests:

```powershell
dotnet test tests/TradingEngine.Tests.Simulation
# → Creates eurusd-h1-*.csv, usdjpy-h1-*.csv in tests/data/
```

## Architecture

```
src/
  TradingEngine.Domain/       # Pure domain types, interfaces, events
  TradingEngine.Application/  # Assembly marker (future use cases)
  TradingEngine.Infrastructure/ # EF Core, Skender, adapters, persistence
  TradingEngine.Risk/         # Risk engine, position sizing, prop firm rules
  TradingEngine.Strategies/   # Strategy implementations
  TradingEngine.Services/     # PipCalculator, SL/TP, trailing stop
  TradingEngine.Host/         # Console + Windows Service host
  TradingEngine.Web/          # ASP.NET Core Razor Pages viewer
  TradingEngine.Adapters.CTrader/ # C# 6 cBot (cTrader integration)
```

## Tech Stack

- **Runtime:** .NET 10, C# 13
- **Persistence:** EF Core + SQLite
- **Indicators:** Skender.Stock.Indicators
- **Logging:** Serilog
- **Testing:** xUnit + FluentAssertions + NSubstitute
- **Web:** ASP.NET Core Razor Pages + Chart.js
